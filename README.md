
## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 打包

```bash
## MacOS
pyinstaller main.py --add-data "static/*:static/" # 打包成文件夹

## Windows
pyinstaller main.py --add-data "static/*;static/" # 打包成文件夹
```

## 配置位置（包括URL）

```
static/config_dev.yml 
```

## ENV设置

```bash
export DY_LIVE_ENV="prod" ## 生效配置 config-<DY_LIVE_ENV>.yml
```
